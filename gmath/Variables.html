Formula Parsing VI Variables

The Formula Parsing VIs accept only the following variables:

Note  Use these same variables to specify the formula string description for the Nonlinear Curve Fit VI.

a, a0, ..., a9

b, b0, ..., b9

.

.

.

z, z0, ..., z9

For variable and function names, only lowercase letters are allowed. The VIs interpret capital letters as errors.

All numbers in exponential notation use the 1E–1 convention with the capital letter E. Using 1e–1 with the lowercase letter e results in an error message.
