Spike Function VI

Owning Palette: Gating Functions VIs

Requires: Full Development System

Generates the spike function for any real number x.

Details  

 Add to the block diagram  Find on the palette
x is any real number.
spike(x) is the value of spike(x) for the given value of x.

Spike Function Details

The following equation defines the spike function:

spike(x) = 1, if 0 x < 1 and 0 elsewhere.

Note  You can define more complex functions by varying and combining the Step Function, Spike Function, and Square Function VIs, respectively.
