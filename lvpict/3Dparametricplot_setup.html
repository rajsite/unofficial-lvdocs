3D Parametric Plot Setup VI

Requires: Full Development System

Plots a surface in terms of x, y, and z points. This VI has three 2D array inputs that specify each point on the plot.

Note  This VI appears on the block diagram when you add a 3D Parametric Graph to the front panel. You can find the 3D Parametric Graph on the 3D Graph palette

color matrix maps each point in the z matrix to a color index on the color ramp. By default, the values of the z matrix are used as the index.
plots in is a reference to the class that stores the data of the 3D plot. This input allows you to wire multiple plots.
x matrix is a 2D array that specifies the x-coordinates of the points on the plot.
y matrix is a 2D array that specifies the y-coordinates of the points on the plot.
z matrix is a 2D array that specifies the z-coordinates of the points on the plot.
plot ID specifies the index of the plot you want to graph.
plots out is the graph of the 3D plot.
