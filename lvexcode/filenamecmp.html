FileNameCmp (LabVIEW Manager Macro)

int32 FileNameCmp(s1, s2);

Purpose

Lexically compares two filenames to determine whether one is less than, equal to, or greater than the other. This comparison uses the same case sensitivity as the file system, that is, case-insensitive on Mac OS X and Windows, case-sensitive on Linux.

Parameters

Name Type Description
s1 PStr Pointer to a Pascal string.
s2 PStr Pointer to a Pascal string.

Return Value

<0, 0, or >0 if s1 is less than, equal to, or greater than s2, respectively. Returns <0 if s1 is an initial substring of s2.
