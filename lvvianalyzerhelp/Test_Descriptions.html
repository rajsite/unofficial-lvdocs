VI Analyzer Test Descriptions (VI Analyzer Toolkit)

Note  The VI Analyzer tests are available only if you install and activate the LabVIEW VI Analyzer Toolkit. Refer to the NI website for more information about the VI Analyzer Toolkit.

The VI Analyzer - Select Tests dialog box contains a list of tests you can run. When you click a test name in the Test hierarchy, a description of the test appears in the text box below the list. The VI Analyzer organizes the tests into the following categories.

Block Diagram tests

Complexity Metrics tests

Documentation tests

Front Panel tests

General tests

Real-Time Module tests

VI Metrics tests

User-Specified tests
