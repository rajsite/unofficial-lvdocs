Using the VI Analyzer Toolkit

The LabVIEW VI Analyzer is a user interface that provides tools for running tests that check VIs interactively for style, efficiency, and other aspects of LabVIEW programming.

Use the VI Analyzer VIs to programmatically check VIs for style, efficiency, and other aspects of LabVIEW programming.

Note  The VI Analyzer functionality this topic describes is available only if you install and activate the LabVIEW VI Analyzer Toolkit or if the LabVIEW Data\VI Analyzer Tests directory contains tests.

Refer to the LabVIEW VI Analyzer Toolkit User Guide for more information about the VI Analyzer toolkit.
