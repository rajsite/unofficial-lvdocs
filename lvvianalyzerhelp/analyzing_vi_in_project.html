Analyzing VIs from the Project Explorer Window

Note  The VI Analyzer functionality this topic describes is available only if you install and activate the LabVIEW VI Analyzer Toolkit or if the LabVIEW Data\VI Analyzer Tests directory contains tests.

You can analyze VIs from an active LabVIEW project if you have saved the VIs and project. Complete the following steps to analyze VIs from the Project Explorer window.

  1. Open a LabVIEW project that includes a saved VI you want to analyze.
  2. From the Project Explorer window, select Tools»VI Analyzer»Analyze VIs to display the VI Analyzer - Select Task dialog box. The VI Analyzer automatically selects the Analyze the current project option.
Note  You also can right-click a VI, folder, target, or project in the Project Explorer window and select Analyze VIs from the shortcut menu to display the VI Analyzer - Select Tests dialog box. Configure the tests you want to run for the current item and click the Analyze button to analyze the VIs.
  1. Click the Next button. The VI Analyzer - Select VIs dialog box appears.
  2. Select the VIs you want to analyze. Click the Next button. The VI Analyzer - Select Tests dialog box appears.
  3. Select the tests and settings you want for the VI Analyzer task. Click the Next button. The VI Analyzer - Exclude Tests dialog box appears.
  4. Deselect tests you want to exclude from running on the VIs.
  5. (Optional) Click the Save button if you want to save the configuration file.
  6. Click the Analyze button. The Analyzing VIs dialog box displays the progress of the VI Analyzer task. After the analysis runs, the VI Analyzer Results Window appears and displays the results of the analysis.
