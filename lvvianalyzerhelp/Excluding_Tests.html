Excluding Tests from a VI Analyzer Task

Note  The VI Analyzer functionality this topic describes is available only if you install and activate the LabVIEW VI Analyzer Toolkit or if the LabVIEW Data\VI Analyzer Tests directory contains tests.

When you create or edit a VI Analyzer task, you can customize it by specifying exactly which tests you want to include or exclude. By default, the VI Analyzer runs all tests on all VIs you select in the VI Analyzer - Select VIs dialog box. You can exclude tests from an entire VI Analyzer task, or you can exclude tests for certain VIs within the task.

Use the VI Analyzer - Select Tests dialog box to include or exclude tests from all VIs in a VI Analyzer task. Place or remove checkmarks in the checkboxes next to the tests you want to include or exclude. You can include or exclude entire categories or specific tests within a category.

Use the VI Analyzer - Exclude Tests from VIs dialog box to exclude tests for a specific VI. Click any VI in the Items to Analyze tree to highlight it. Place or remove checkmarks in the checkboxes next to the tests you want to include or exclude for that VI.

Refer to the Customizing Tests within VI Analyzer Tasks section of the LabVIEW VI Analyzer Toolkit User Guide for more information about including and excluding tests from VI Analyzer tasks.
