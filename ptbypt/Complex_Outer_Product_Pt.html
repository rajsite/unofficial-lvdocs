Complex Outer Product PtByPt VI

Owning Palette: Linear Algebra PtByPt VIs

Requires: Full Development System

Computes the Outer Product of the complex values y and x.

This VI is similar to the Outer Product VI.

Note  By default, reentrant execution is enabled in all Point By Point VIs.

 Add to the block diagram  Find on the palette
initialize, when TRUE, initializes the internal state of the VI.
y is a complex-valued data point.
sample length y is the length of each set of incoming data. The VI performs computation for each set of data. The default is 10. sample length y must be greater than zero.
x is a complex-valued data point.
sample length x is the length of each set of incoming data. The VI performs computation for each set of data. The default is 10. sample length x must be greater than zero.
Outer Product is the outer product computed from the complex values.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.
