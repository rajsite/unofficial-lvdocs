Convolution PtByPt VI

Owning Palette: Signal Operation PtByPt VIs

Requires: Full Development System

Computes the convolution of x and y.

This VI is similar to the Convolution VI.

Note  By default, reentrant execution is enabled in all Point By Point VIs.

 Add to the block diagram  Find on the palette
initialize, when TRUE, initializes the internal state of the VI.
x is an input data point.
sample length x is the length of each set of incoming x-data. The VI performs computation for each set of data. The default is 100. sample length x must be greater than zero.
y is an input data point.
sample length y is the length of each set of incoming y-data. The VI performs computation for each set of data. The default is 100. sample length must be greater than zero.
X*Y is the convolution of x and y.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.
