Trace of a Matrix

The trace of a square matrix is the sum of the diagonal elements of the matrix. For example, if

The trace of a matrix is the result of a linear transform from the matrix space to the scalar space. If A and B are two square matrices, then trace(A + B) = trace(A) + trace(B), trace(a A) = a trace(A), where a is a scalar.

The following are some other important properties of the trace of a matrix:

  • The trace of a matrix equals the trace of the transpose of the matrix. In other words, trace(A) = trace(AT).
  • The trace of a matrix is similarity invariant, as shown in the following equation: trace(A) = trace(SAS-1), where S is an invertible matrix.
  • trace(A B) equals trace(B A).
  • The trace of a matrix equals the sum of its eigenvalues.
