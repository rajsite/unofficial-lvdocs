Interpolation

In some applications, a function y = f(x) is usually given at some discrete points, such as (x0, y0), (x1, y1), …, (xn - 1, yn - 1).

To find the value of y at the value of x, you construct the function that closely fits those data points. In the case of interpolation, a type of curve fitting, the function must fit the given data points exactly.
