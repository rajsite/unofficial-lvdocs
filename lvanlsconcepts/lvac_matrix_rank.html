Matrix Rank

The rank of a matrix A, denoted by ρ(A), is the maximum number of linearly independent columns in A. The following list describes some useful properties of the rank of a matrix:

  • The rank is invariable to the transpose. In other words, rank(A) = rank(AT).
  • If one or more solutions exist to the linear system Ax = b, then rank([A, b]) must equal rank(A), where the matrix [A, b] is the augmented matrix.
  • Elementary operations do not change the rank of the matrix.
  • For any two m-by-n matrices A and B, when there are two nonsingular square matrices X and Y that satisfy B = XAY, the rank of matrix A equals the rank of matrix B.
