Confidence Interval

You can use curve fitting to find the coefficients of a model that describes the relationship between variables. In real-world applications, you do not know the exact values of the actual coefficients. Instead, the model coefficients you calculate from the data set represent guesses for the actual coefficients. Typically, these guesses are different from the actual coefficients due to errors in measurement. You can use the confidence interval to determine how accurate the guesses are and to evaluate the quality of the curve fitting.

To calculate the confidence interval, the measurement error must be Gaussian distributed. Also, you must use the least squares method to fit the curve.

You calculate the confidence interval using two steps. You first calculate the confidence interval of the coefficients. You then calculate the confidence interval of the data points.

The following figure shows an example of the confidence interval in linear fitting. The black line is the fitting line calculated using the least squares method from the measured data set. The area between the confidence bounds is the confidence interval. In this example, the confidence level is 0.95, which means there is a 95% probability that the actual data points, and thus the actual curve, fall within the confidence interval. Because the measurement error on each data point is not the same, the blue confidence bounds are not linear like the black fitting line.



For a given confidence level, the narrower the confidence interval, the more likely the fitted curve is closer to the actual curve. The confidence interval also reflects the quality of the measurements. If the measurements contain a lot of noise, the confidence interval is larger.
