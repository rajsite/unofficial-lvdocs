The Finite Element Method

Unlike the finite difference method, which approximates the partial derivative in the equation by finite difference, the finite element method approximates the solution with a linear combination on a set of basis functions. When the domain separates into small pieces, it generates discrete elements, which define the basis functions. One helpful feature of this method is the straightforward handling of geometrically complicated domains, such as the L-shaped domain.
