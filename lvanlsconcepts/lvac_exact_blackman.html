Exact Blackman

The following equation defines the Exact Blackman window.

for n = 0, 1, 2, …, N - 1 and ω = 2πn/N

where N is the length of the window
  a0 = 7938/18608
  a1 = 9240/18608
  a2 = 1430/18608.

The Exact Blackman window is useful for single tone measurement. The Exact Blackman window has a lower main lobe width and a lower maximum side lobe level than the Blackman window. However, the Blackman window has a higher side lobe roll-off rate than the Exact Blackman window.
