Variable Step-Size ODE Solvers versus Fixed Step-Size ODE Solvers

Variable step-size ODE solvers adjust the step size throughout the calculation to ensure that this per-step error remains at a given relative and absolute tolerance.

For each integrator variable y, the ODE solver varies the step size to control the error according to the following approximation:

per-step error ≈ |y| * relative tolerance + absolute tolerance

Variable step-size ODE solvers can take small time steps when the variables vary rapidly and can take larger time steps when the variables vary slowly. This ability to change step sizes can increase computational efficiency.

Variable step-size ODE solvers are not appropriate for deterministic real-time applications because the computational overhead of taking a time step varies over the course of an application. You can use fixed step-size, deterministic ODE solvers for real-time applications. These ODE solvers do not estimate the local per-step error and maintain a fixed step size throughout a calculation.
