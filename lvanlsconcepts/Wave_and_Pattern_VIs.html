Signal Generation in LabVIEW

The Signal Generation VIs create many common signals required for network analysis and simulation. You also can use the Signal Generation VIs in conjunction with National Instruments hardware to generate analog output signals.

The names of most of the Signal Generation VIs contain the word wave or pattern. These names indicate whether a VI can keep track of the phase of the signal it generates each time the VI is called.

Note   You can use the Signal Generation VIs only in the LabVIEW Full and Professional Development Systems.

When you use a VI that requires the normalized frequency as an input, you must convert the frequency units to the normalized units of cycles per sample. You must use normalized units of cycles per sample with the following Signal Generation VIs:

  • Sine Wave VI
  • Square Wave VI
  • Sawtooth Wave VI
  • Triangle Wave VI
  • Arbitrary Wave VI
  • Chirp Pattern VI
