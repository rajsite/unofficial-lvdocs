Cholesky Factorization

If the input matrix A is symmetric and positive definite, an LU factorization can be computed such that A = UTU, where U is an upper triangular matrix. This is Cholesky factorization.

For a complex Hermitian positive definite matrix A, its Cholesky factorization is A = RHR, and the diagonal elements of R are real and positive. If the Cholesky factorization exists, the matrix R is unique. You can regard the matrix R as the square root of matrix A.

You can use Cholesky factorization to determine whether a symmetric matrix is positive definite.
