The Kronecker product satisfies the following properties. You can use these properties to solve matrix equations with the Kronecker product.

  • (αA) ⊗ B = A ⊗ (αB), where α is a scalar
  • (A ⊗ B)H = AH⊗ BH, where .H is the conjugated transpose
  • (A ⊗ B) ⊗C = A ⊗ (B ⊗ C), (A ⊗ B) (C ⊗ D) = AC ⊗ BD
  • (A + B) ⊗ C = A ⊗ C + B ⊗ C, A ⊗ (B + C) = A ⊗ B + A ⊗ C
  • (A ⊗ B)-1 = A-1 ⊗ B-1
  • trace(A ⊗ B) = trace(A) · trace(B)
