Accessing LabVIEW Add-ons with the VI Package Manager (VIPM) Software (Windows)

NI uses the JKI VI Package Manager Free Edition (VIPM) software to provide access to LabVIEW add-ons and other code distributed as JKI VI Package files. The VIPM software allows you to find, install, and manage LabVIEW add-ons distributed on the LabVIEW Tools Network at ni.com/labview-tools-network.

The JKI VIPM software supports installing and managing add-ons distributed as VI Package (.vip) files. VI Packages are similar to zip files in that they contain all the VIs and resources for a specific LabVIEW add-on, as well as information the VIPM software uses to install, remove, and manage the add-on.

To launch the VIPM software if it is installed, select Find Drivers and Add-ons»Find LabVIEW Add-ons from the Getting Started window. If the VIPM software is not installed, selecting Find Drivers and Add-ons»Find LabVIEW Add-ons from the Getting Started Window opens the LabVIEW Tools Network.

Refer to the JKI website at jki.net for more information about the VIPM software. Refer to the VIPM software documentation for information about using the VIPM software to find, install, and manage add-ons.
