Show in File System VI

Owning Palette: Advanced File VIs and Functions

Requires: Base Development System

Opens a path to a file or directory in (Windows) Windows Explorer, (macOS) the Finder, or (Linux) a file system browser depending on the current platform. If you specify a path to a file that is inside an LLB or a packed project library, this VI opens the path to the LLB or the packed project library.

file or directory path specifies the path to a file or directory.
