Include Fixed-Point Overflow Status Function

Owning Palette: Fixed-Point Functions

Requires: Base Development System

Includes an overflow status with FXP. The value of the overflow status equals the value specified in overflow.

 Add to the block diagram  Find on the palette
FXP specifies a fixed-point number.
overflow specifies whether the overflow status of FXP with overflow included is TRUE or FALSE. If overflow is TRUE, the overflow status of FXP with overflow included is TRUE. Otherwise, the overflow status of FXP with overflow included is FALSE. The default is FALSE.
FXP with overflow included returns FXP with an overflow status set to the value specified in overflow.
