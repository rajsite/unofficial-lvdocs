List Error Cluster Attributes VI

Owning Palette: Multiple Errors VIs

Requires: Base Development System

Returns the names of all attributes stored in an error cluster. For a single cluster that stores multiple errors, this VI lists only the attribute names of the first error.

 Add to the block diagram  Find on the palette
error specifies an error cluster that you use to check for errors or warnings.
attribute names returns the names of all attributes stored in the error cluster.
