Get Waveform Information VI

Owning Palette: Data Type Parsing VIs

Requires: Base Development System

Retrieves waveform information from the data type stored in variant. This VI returns an error if variant does not contain a waveform.

 Add to the block diagram  Find on the palette
variant specifies the variant data from which you want to retrieve data type information.
error in describes error conditions that occur before this node runs. This input provides standard error in functionality.
Y array type returns the data type of the Y array element of the waveform stored in variant.
error out contains error information. This output provides standard error out functionality.
