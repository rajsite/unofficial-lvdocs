Assert Integer Type VI

Owning Palette: Assert Type VIs and Functions

Requires: Base Development System

Breaks the calling VI unless the input data is an integer numeric type.

This VI does nothing at run time. Use this VI in conjunction with the Type Specialization structure to customize sections of code in a malleable VI (.vim) for integer numeric types or to force a malleable VI to accept integer numeric types only.

 Add to the block diagram  Find on the palette
value in specifies the input data.

This input accepts any integer numeric data type.
value out returns the input data if value in is an integer numeric type. Otherwise, this output returns a 32-bit signed integer 0.
