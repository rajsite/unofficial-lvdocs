Merge Query Expressions VI

Owning Palette: Advanced Storage VIs

Requires: Base Development System (Windows)

Merges two query expressions. You can use the Create Query Expression VI to create query expressions that you want to merge.

 Add to the block diagram  Find on the palette
query expression 1 specifies a query expression you want to merge.
query expression 2 specifies a query expression you want to merge.
error in describes error conditions that occur before this node runs. This input provides standard error in functionality.
combined query expression returns a query expression that merges query expression 1 and query expression 2. You can wire this output to the Execute Query Expression VI to obtain the objects that meet the combined query conditions.
error out contains error information. This output provides standard error out functionality.
