Get Help Window Status Function

Owning Palette: Help VI and Functions

Requires: Base Development System

Returns the status and the position of the Context Help window.

(Real-Time Module) You cannot use this function in VIs that run on RT targets.

 Add to the block diagram  Find on the palette
If Show is TRUE, the Context Help window is visible. If FALSE, the Context Help window is not visible.
Top Left Corner indicates the position of the Context Help window and contains numbers that represent points offset from the upper left corner of the screen.
Horizontal is the number of points that the Context Help window is to the right of the upper left corner of the screen.
Vertical is the number of points that the Context Help window is below the upper left corner of the screen.
