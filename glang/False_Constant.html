False Constant

Owning Palette: Boolean Functions

Requires: Base Development System

Use this constant to provide a value of FALSE to the block diagram.

 Add to the block diagram  Find on the palette

You can assign a label to this constant by right-clicking the constant and selecting Visible Items»Label from the shortcut menu.

You can change this value to TRUE by using the Operating tool to click the F portion of the constant or by right-clicking the constant and selecting Data Operations»Change Value to True from the shortcut menu. You cannot change this value while the VI runs.
