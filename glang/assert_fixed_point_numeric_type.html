Assert Fixed-Point Numeric Type VI

Owning Palette: Assert Type VIs and Functions

Requires: Base Development System

Breaks the calling VI unless the input data is a fixed-point numeric type.

This VI does nothing at run time. Use this VI in conjunction with the Type Specialization structure to customize sections of code in a malleable VI (.vim) for fixed-point numeric data types or to force a malleable VI to accept fixed-point numeric types only.

 Add to the block diagram  Find on the palette
value in specifies the input data.
value out returns the input data if value in is a fixed-point numeric type. Otherwise, this output returns a fixed-point number 0.
