Power Of 2 Function

Owning Palette: Exponential Functions

Requires: Base Development System

Computes 2 raised to the x power.

The connector pane displays the default data types for this polymorphic function.

Example

 Add to the block diagram  Find on the palette
x can be a scalar number, array or cluster of numbers, array of clusters of numbers, and so on.
2^x is of the same numeric representation as x. When x is of the form x = a + bi, that is, when x is complex, the following equation defines 2^x:

2^x = 2^a * 2^(bi) = 2^a * (cos(b * ln(2)) + i sin(b * ln(2)))
