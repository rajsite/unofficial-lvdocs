To Time Stamp Function

Owning Palette: Conversion VIs and Functions

Requires: Base Development System

Converts a number to a timestamp.

 Add to the block diagram  Find on the palette
number can be a scalar number, array or cluster of numbers, array of clusters of numbers, and so on, that represents the amount of seconds from the epoch time of 12 a.m., January 1, 1904 [01-01-1904 00:00:00].
time stamp is a time-zone-independent number of seconds that have elapsed since 12:00 a.m., Friday, January 1, 1904, Universal Time [01-01-1904 00:00:00].
