Local Variable Object Reference

Owning Palette: Shared Variable Node, VI, and Functions

Requires: Base Development System

Use this constant to supply a reference to one of the following items: local PSP Variable Engine, local I/O Variable Engine, local Root PSP Variable Container, or local Root I/O Variable Container.

 Add to the block diagram  Find on the palette

To define the type of the constant, click inside the border of the constant and select the type from the shortcut menu.

Use this constant when finding variables programmatically.
