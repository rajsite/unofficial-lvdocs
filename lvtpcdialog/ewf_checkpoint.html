EWF Checkpoint VI

Owning Palette: Enhanced Write Filter VIs

Requires: Application Builder or Professional System

Creates a new checkpoint on a volume. The volume must be enabled for enhanced write filter (EWF). This VI is only valid when the volume is in EWF disk mode.

Note  You must restart the operating system for changes to take effect.

Example

 Add to the block diagram  Find on the palette
volume name specifies the name of the volume to create a checkpoint. For example, enter C: to specify the C drive.
error in describes error conditions that occur before this node runs. This input provides standard error in functionality.
overlay level returns the overlay level of the volume whose checkpoint this VI creates.
error out contains error information. This output provides standard error out functionality.
