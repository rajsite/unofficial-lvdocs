EWF Restore VI

Owning Palette: Enhanced Write Filter VIs

Requires: Application Builder or Professional System

Restores a volume to the last checkpoint and removes all changes made after that checkpoint. The volume must be enabled for enhanced write filter (EWF).

Note  You must restart the operating system for changes to take effect.

Example

 Add to the block diagram  Find on the palette
volume name specifies the name of the volume whose checkpoint you want to restore. For example, enter C: to specify the C drive.
error in describes error conditions that occur before this node runs. This input provides standard error in functionality.
overlay level returns the overlay level whose volume this VI restored.
error out contains error information. This output provides standard error out functionality.
