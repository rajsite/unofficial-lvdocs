NI Distributed System Manager (Windows)

Select Tools»Distributed System Manager to display the NI Distributed System Manager. Use the System Manager to create, edit, monitor, and delete shared variables outside of the project environment. You also can manage the lifetime of the Shared Variable Engine with this dialog box.

The System Manager also allows you to manage the LabVIEW Web Server and LabVIEW Web services. You can pause, resume, delete, and restart individual Web Servers and Web services, as well as view statistics for Web services using the System Manager. Refer to the NI Distributed System Manager Help for information about managing Web servers and Web services using the System Manager.
