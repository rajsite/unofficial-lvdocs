Connection Status Property

Property of DataSocket.

Returns a cluster that contains the status and error code of a data connection. The connection status can be Invalid, Unconnected, ConnectionActive, ConnectionIdle, ConnectionError, or Connecting. If the status is ConnectionError, this property also returns the error code.
