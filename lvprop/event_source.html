Event Reference

Use this book to learn about events in LabVIEW. This book contains information about Control, Application, Pane, and VI class events. The following table describes the source of LabVIEW events.

0LabVIEW UI—All user interface events that LabVIEW generates internally.
1ActiveX—All events that ActiveX generates.
2.NET—All events that .NET generates.
25User Event—All events that the user creates programmatically.
