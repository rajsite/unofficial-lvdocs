Fuzzy System Designer Menus

Select Tools»Control Design and Simulation»Fuzzy System Designer to display the Fuzzy System Designer. The Fuzzy System Designer includes the following menus:

The File menu includes the following items:

  • New—Creates a new, blank fuzzy system.
  • Open—Displays a standard file dialog box you can use to navigate to and open a .fs file for a fuzzy system.
  • Import—Imports a .fc file.
  • Save—Saves the current fuzzy system. If you are saving a new fuzzy system for the first time, a dialog box prompts you to name the file and determine its location.
  • Save As—Saves a copy of the current fuzzy system, renames the fuzzy system, or copies the fuzzy system to a new location.
  • Description—Launches the Edit Description dialog box, with which you can enter a description for the fuzzy system. This description appears in the HTML report of the fuzzy system that you can create by selecting File»Create HTML Report.
  • Create HTML Report—Creates an HTML report that summarizes all parts of the fuzzy system.
  • Close—Closes the current fuzzy system. A confirmation dialog box prompts you to save any changes you have made to the fuzzy system.

The Operate menu includes the following items:

  • Clear All Rules—Deletes all existing rules from the Rules page of the Fuzzy System Designer.
  • Pre-Generate Rules—Launches the Pre-Generate Rules dialog box, with which you can generate a complete rule base for the fuzzy system.

The Help menu includes the Show Context Help item, which displays or hides the Context Help window.
