Blackman-Harris Window VI

Owning Palette: Windows VIs

Requires: Full Development System

Applies a three-term, Blackman-Harris window to the input signal X. Wire data to the X input to determine the polymorphic instance to use or manually select the instance.

Details  Example

Use the pull-down menu to select an instance of this VI.

Blackman-Harris Window (DBL)

X is a real vector.
Windowed X is the input signal with the window applied.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.

Blackman-Harris Window (CDB)

X is the complex valued input sequence.
Windowed X is the input signal with the window applied.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.

Blackman-Harris Window Details

If Y represents the output sequence Windowed X, the Blackman-Harris Window VI obtains the elements of Y using the following formula:

yi = xi[0.42323 – 0.49755cos(w) + 0.07922cos(2w)]

for i = 0, 1, 2, …, n – 1

where n is the number of elements in X and

.
