Flat Top Window VI

Owning Palette: Windows VIs

Requires: Full Development System

Applies a flat top window to the input sequence X. Wire data to the X input to determine the polymorphic instance to use or manually select the instance.

Details  Example

Use the pull-down menu to select an instance of this VI.

Flat Top Window (DBL)

X is a real vector.
Windowed X is the input signal with the window applied.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.

Flat Top Window (CDB)

X is the complex valued input sequence.
Windowed X is the input signal with the window applied.
error returns any error or warning from the VI. You can wire error to the Error Cluster From Error Code VI to convert the error code or warning into an error cluster.

Flat Top Window Details

If y represents the output sequence Windowed X, the Flat Top Window VI obtains the elements of y from

yi = xi[0.21557895 – 0.41663158cos(w) + 0.277263158cos(2w) – 0.083578947cos(3w) + 0.006947368cos(4w)]

for i = 0, 1, 2, …, n – 1

,

where n is the number of elements in X.
