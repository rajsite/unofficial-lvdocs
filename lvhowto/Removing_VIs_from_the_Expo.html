Removing VIs from the Exported VIs List

Complete the following steps to remove a VI name or directory path from the Exported VIs list.

  1. Select Tools»Options to display the Options dialog box.
  2. Select VI Server from the Category list.
  3. In the Exported VIs section, select the VI name or directory path in the Exported VIs list you want to remove.
  4. Click the Remove button.
  5. Click the OK button to close the Options dialog box and save the changes.
