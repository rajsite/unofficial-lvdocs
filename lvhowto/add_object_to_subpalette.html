Adding VIs or Objects to the Functions or Controls Palette

Add VIs and objects to the Functions or Controls palette or subpalettes to use in a new or existing VI. A subpalette can contain VIs, functions, controls, and subpalettes. A subpalette can also contain VIs from different locations.

Adding VIs or Functions to the Functions Palette

You can add any VI block diagram or custom function to the Functions palette. You can then place the block diagram contents of one VI on the block diagram of another VI straight from the Functions palette instead of calling a subVI.

Complete the following steps to add a VI or function to the Functions palette:

  1. Create a VI or function you want to add to the Functions palette. Save and close the file.
  2. Select Tools»Advanced»Edit Palette Set to display the Edit Controls and Functions Palette Set dialog box.
  3. Navigate to the location on the Functions palette where you want to add the VI or function.
  4. Right-click the palette and select Insert»VI(s) from the shortcut menu.
  5. Navigate to the VI or function you want to add to the palette.
  6. Select the VI or function you want to add to the palette and click Open.
  7. Right-click the VI or function icon and select Place VI Contents from the shortcut menu.
  8. Click Save Changes in the Edit Controls and Functions Palette Set dialog box to open the Preview Palette Changes dialog box.
  9. Click Continue to accept the changes and close the Edit Controls and Functions Palette Set dialog box.

Adding VIs or Controls to the Controls Palette

You can add any VI front panel or custom control to the Controls palette. Unlike adding custom controls, adding VIs allows you to add multiple controls or decorations to the front panel at a time.

Note  To add custom decorations to the Controls palette, create a VI and add the decorations to the front panel. Then add the VI to the Controls palette.

Complete the following steps to add a VI or control to the Controls palette:

  1. Create a VI or control you want to add to the Controls palette. Save and close the file.
  2. Select Tools»Advanced»Edit Palette Set to display the Edit Controls and Functions Palette Set dialog box.
  3. Navigate to the location on the Controls palette where you want to add the VI or control.
  4. Right-click the palette and select Insert»Custom Control(s) from the shortcut menu.
  5. Navigate to the VI or control you want to add to the palette.
    Note  You may have to select All Files (*.*) from the file type pull-down menu to include the .vi file extension in your search.
  6. Select the VI or control you want to add to the palette and click Open.
  7. Click Save Changes in the Edit Controls and Functions Palette Set dialog box to open the Preview Palette Changes dialog box.
  8. Click Continue to accept the changes and close the Edit Controls and Functions Palette Set dialog box.
