Searching PDF Versions of LabVIEW Manuals

Use Adobe Reader 7.0 or later to search PDF versions of the LabVIEW manuals. Refer to the Adobe Systems Incorporated Web site to download Adobe Reader.

Complete the following steps to search all the PDF documents that ship with LabVIEW.

  1. In Adobe Reader, select Edit»Advanced Search to display the Search window.
  2. Enter a word or phrase in the What word or phrase would you like to search for? text box.
  3. Click the All PDF Documents in button and select Browse for Location from the pull-down menu. The Browse for Folder dialog box appears.
    1. Navigate to the labview\manuals directory.
    2. Click the OK button to close the dialog box and return to the Search window.
  4. Click the Search button.

Refer to the Adobe Reader Help for more information about searching all the PDF documents in a directory for a word or phrase.
