Changing the Numeric Representation of a Waveform

The waveform data (Y) of the waveform control can be an array of any numeric data type other than the fixed-point numeric data type. The interior of the waveform terminal appears orange if the data consists of floating-point or complex numbers and blue if the data consists of integers.

Complete the following steps to change the representation of the Y array of a waveform.

  1. On the front panel or block diagram, right-click the waveform object.
  2. Select Y Array Representation from the shortcut menu.
  3. Select the data type that best represents the data.
