Logging In and Logging Out of LabVIEW (Windows)

Complete the following steps to log in to LabVIEW.

  1. Select Tools»Security»Login to display the NI Security Login dialog box.
  2. Enter the user account name in the User name text box.
  3. Enter the account password in the Password text box.
  4. Select the security domain in which the user account is defined from the NI domain listbox.
Note  If there are no security domains to choose from, you can define one using the Domain Account Manager.
  1. Click OK to log in.

To log out of LabVIEW, select Tools»Security»Logout.

Note  When no one is logged in, LabVIEW uses a default nobody account to define the access rights for the user. There is no password assigned to the nobody account.
