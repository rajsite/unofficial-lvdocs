Selecting Items in Ring and Enumerated Type Controls at Run Time

Use the following ways to select items in a ring or enumerated type control while a VI is in run mode:

  • Click the increment or decrement buttons.
  • Press the <Tab> key to navigate to the control and press the up and down arrow keys.

Use the following ways to select items in a ring or enumerated type control while a VI is in run mode or to select items in a ring or enum constant while a VI is in edit mode:

  • Click the control to display the shortcut menu and use the up and down arrow keys to navigate to the item you want to select. After you highlight the item in the shortcut menu, press the <Enter> key to select the item.
  • Click the control to display the shortcut menu and type the beginning letters of the string you want to select. LabVIEW highlights the first item in the control that begins with the letters you type. Press the <Enter> key to select the item.
  • If the digital display is visible, place the cursor in the digital display, enter a numeric value, and press the <Enter> key. If you enter a value that does not exist in the control, LabVIEW changes the value you entered to the closest valid value.
  • If a ring control allows undefined numeric values, click the control, select <Other> from the shortcut menu, enter a numeric value in the digital display that appears, and press the <Enter> key. The undefined value appears in the ring control in angle brackets. If the digital display is visible, you also can place the cursor in the digital display, enter a numeric value, and press the <Enter> key.
