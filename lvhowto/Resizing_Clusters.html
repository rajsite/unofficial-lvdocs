Resizing Clusters

Complete the following steps to shrink clusters to fit their contents.

  1. Right-click a cluster shell border and select AutoSizing»Size to Fit from the shortcut menu.
  2. Add another element to the cluster shell. When AutoSizing»Size to Fit is selected, the cluster resizes itself if necessary when you add another element to the cluster shell.
  3. Right-click the cluster shell border and select AutoSizing»Arrange Horizontally or AutoSizing»Arrange Vertically from the shortcut menu to arrange the elements in the cluster horizontally or vertically.
