Editing the Shortcut Menus of a Polymorphic VI

Complete the following steps to edit the instance names to display in the shortcut menus of a polymorphic VI you created and its selector.

Note  Use caution when editing polymorphic VIs that ship with LabVIEW.
  1. Right-click a polymorphic VI on the block diagram and select Open Polymorphic VI from the shortcut menu to display the Polymorphic VI window. You also can select File»Open and open a polymorphic VI to display the window.
  2. Select the instance whose name you want to edit in the Instance VIs list.
  3. Click the Edit Name button to display the Edit Instance Name dialog box.
  4. In the Menu Name text box, enter the name to use in the polymorphic VI shortcut menu. To create a hierarchical shortcut menu, use a colon (:) to separate menu items.
  5. In the Polymorphic VI Selector Name text box, enter the name to use in the polymorphic VI selector shortcut menu. To create a multiline selector name, press the <Enter> key on the keyboard to start a new line.
  6. Click the OK button to return to the Polymorphic VI window. The shortcut menu names appear next to the instance VI name in the Instance VIs list.
  7. Select File»Save to save the VI.
  8. Select File»Close to close the Polymorphic VI window.
