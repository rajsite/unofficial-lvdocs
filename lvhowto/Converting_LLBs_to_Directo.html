Converting LLBs to Directories

Complete the following steps to convert LLBs to directories when you want to remove VIs from an LLB and have them appear as separate VIs.

  1. Select Tools»LLB Manager to display the LLB Manager window.
  2. Navigate to the directory where the LLB resides by clicking the down arrow in the pull-down menu. LLBs, other files, and subdirectories appear in the list below the pull-down menu.
  3. Select the LLB to convert.
  4. Click the Convert Directories and LLBs button to display the Conversion Options dialog box.
  5. (Optional) Configure the Directory name and Original LLB options.
  6. Click the Convert button.
  7. Select File»Close to exit the LLB Manager window.

You also can convert directories to LLBs.
