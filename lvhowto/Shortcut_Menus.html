Disabling Run-Time Shortcut Menus

Complete the following steps to disable all run-time shortcut menus for a VI so users cannot right-click objects and change their appearance.

  1. Select File»VI Properties to display the VI Properties dialog box.
  2. Select Window Appearance from the Category pull-down menu.
  3. Click the Customize button to display the Customize Window Appearance dialog box.
  4. Remove the checkmark from the Allow default run-time shortcut menus checkbox.
  5. Click the OK button twice to save the changes.

To disable the run-time shortcut menus for an individual control or indicator, right-click the object and select Advanced»Run-Time Shortcut Menu»Disable from the shortcut menu.
