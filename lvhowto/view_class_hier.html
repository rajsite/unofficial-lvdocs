Viewing the Inheritance Hierarchy of a LabVIEW Class or Interface

The LabVIEW Class Hierarchy window displays a graphical representation of the inheritance of LabVIEW classes and interfaces. Use this window to view the hierarchy of the classes or interfaces currently in memory. Complete the following steps to use the LabVIEW Class Hierarchy window.

  1. Right-click the class or interface whose hierarchy you want to view and select Show Class Hierarchy. The LabVIEW Class Hierarchy window appears.
  2. Select from the following options:
    • Move the cursor over the classes or interfaces in the LabVIEW Class Hierarchy window to view the names of the classes or interfaces.
    • Search the LabVIEW class hierarchy by typing the name of the class or interface you want to find anywhere in the window. The search string appears and displays the text as you type. LabVIEW highlights one item at a time whose name begins with the search string.
    • View the hierarchy of all classes and interfaces in the LabVIEW project by selecting View»Show All Classes in the LabVIEW Class Hierarchy window.
    • Use the buttons on the LabVIEW Class Hierarchy window toolbar to select more ways to view the LabVIEW class hierarchy.

You also can use the Class Browser window to view the LabVIEW class hierarchy.
