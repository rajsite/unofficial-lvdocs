Removing Visible VIs Entries

Complete the following steps to remove a VI name or directory path from the Visible VIs list.

  1. Select Tools»Options to display the Options dialog box.
  2. Select Web Server from the Category list.
  3. Select the VI name or directory path you want to remove in the Visible VIs list in the Visible VIs section.
  4. Click the Remove button.
  5. Click the OK button to close the Options dialog box and save the changes.
