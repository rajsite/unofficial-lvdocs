Creating Template Controls

Complete the following steps to create a template control.

  1. Right-click a front panel control or indicator that you want to use as a template and select Advanced»Customize from the shortcut menu to display the Control Editor window.
  2. Create a custom control to include only the components you want to reuse.
  3. Select File»Save As to display a file dialog box.
  4. Select Template Controls from the Save as type pull-down menu.
  5. Enter the name of the template control and click the OK button.
Caution   Do not save your own VIs and other files in .llb files installed with LabVIEW.
