Resetting 2D Picture Controls

By default, the 2D picture control clears its display before drawing an image. If you want the 2D picture control to draw the new image over the previous image, right-click the control and select Erase First from the shortcut menu to remove the checkmark from this menu item.

You also can use the Erase First property to specify whether the 2D picture control erases the previous image programmatically.
