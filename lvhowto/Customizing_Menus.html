Customizing Menus

You can create custom menus for every VI you build using the Menu Editor dialog box, and you can configure VIs to show or hide menu bars. You also can associate an existing run-time menu (.rtm) file with a VI.

Note  Custom menus appear only while the VI runs.

You can build custom menus or modify the default LabVIEW menus statically when you edit the VI or programmatically when you run the VI.
