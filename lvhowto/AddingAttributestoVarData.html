Adding Attributes to Variant Data

Complete the following steps to add attributes to variant data.

  1. Add the Set Variant Attribute function to the block diagram.

     Add  Find
  2. Wire the variant data to the variant input of the Set Variant Attribute function.
  3. Wire the name and value inputs of the Set Variant Attribute function.
  4. Wire the variant out output of the Set Variant Attribute function to any VI or function that accepts variant data as an input.
