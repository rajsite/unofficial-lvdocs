Entering or Changing Text in a String Control

Complete the following steps to enter or change text in a string control.

  1. Add a string control to the front panel. The default mode is Normal Display.

     Add  Find
  2. Use the Operating tool or the Labeling tool to add the cursor to the string control and enter or edit the text you want. When the text reaches the right border of the string display, the string wraps to the next line, breaking on a natural separator such as a space or tab character.
Note  You can double-click the text to highlight a word or triple-click the text to highlight the entire string.

You also can use several keyboard shortcuts in a string control.
