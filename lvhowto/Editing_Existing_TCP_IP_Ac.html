Editing Existing Machine Access List Entries

Complete the following steps to edit a browser address in the Machine access list.

  1. Select Tools»Options to display the Options dialog box.
  2. Select VI Server from the Category list.
  3. In the Machine access section, select the address in the Machine access list you want to modify. The address appears in the text box.
  4. Edit the address in the text box or click the Allow Access or Deny Access buttons. You also can use wildcards in the browser address.
  5. Click the OK button to close the Options dialog box and save the changes.
