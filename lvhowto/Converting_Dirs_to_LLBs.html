Converting Directories to LLBs

Convert directories to LLBs when you want to group all the VIs in a directory in an LLB.

Complete the following steps to convert directories to LLBs.

  1. Select Tools»LLB Manager to display the LLB Manager window.
  2. Navigate to the directory where the VIs reside by clicking the down arrow in the pull-down menu. LLBs, other files, and subdirectories appear in the list below the pull-down menu.
  3. Select the directory where the VIs reside.
  4. Click the Convert Directories and LLBs button to display the Directories to Convert dialog box.
  5. Click the Convert button.
  6. Select File»Close to exit the LLB Manager windows.

You also can convert LLBs to directories.
