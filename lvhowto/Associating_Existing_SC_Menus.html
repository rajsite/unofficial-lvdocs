Associating Existing Run-Time Shortcut Menus

You can use an existing run-time shortcut menu (.rtm) file with more than one control.

Complete the following steps to associate an existing .rtm file with a control.

  1. Right-click a control and select Advanced»Run-Time Shortcut Menu»Edit from the shortcut menu to display the Shortcut Menu Editor dialog box.
  2. Select File»Open in the Shortcut Menu Editor dialog box.
  3. Navigate to the shortcut menu file you want to associate with the control and click the OK button. Notice that the path to the menu file is displayed in the title bar of the Shortcut Menu Editor dialog box.
  4. Close the Shortcut Menu Editor dialog box.
  5. When asked if you want to change the run-time shortcut menu to the selected menu file, click the Yes button.

Complete the following steps to disassociate a run-time shortcut menu from a control.

  1. Right-click a control and select Advanced»Run-Time Shortcut Menu»Edit from the shortcut menu to display the Shortcut Menu Editor dialog box.
  2. Select Default from the Menu Type pull-down menu. Notice that a file path no longer appears on the title bar of the Shortcut Menu Editor dialog box.
  3. Close the Shortcut Menu Editor dialog box.
  4. When asked if you want to change the run-time shortcut menu to the Default menu, click the Yes button.

You can customize run-time shortcut menus statically and programmatically.
