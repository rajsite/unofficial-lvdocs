Determining the Size of Arrays

Complete the following steps to determine the size of an array.

  1. Right-click the array index display.
  2. Select Advanced»Show Last Element from the shortcut menu. The index display shows the dimension of the array, thus giving you the size of the array. For example, if you have an array of 15 rows and 15 columns and you complete this step, the index display shows 14 in the upper display and 14 in the lower display.

You also can change the data size of an array.
