Editing Existing Exported VIs Entries

Complete the following steps to edit the name of a VI or directory path in the Exported VIs list.

  1. Select Tools»Options to display the Options dialog box.
  2. Select VI Server from the Category list.
  3. Select the VI name or directory path in the Exported VIs list you want to modify. The name or path appears in the text box.
  4. Edit the name or path in the text box or click the Allow access or Deny access buttons. You also can use wildcards in the VI name or directory path.
  5. Click the OK button to close the Options dialog box and save the changes.
