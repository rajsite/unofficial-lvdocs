Setting the Control and Indicator Style for the Current VI

The Editor Options page of the VI Properties dialog box and the Front Panel page of the Options dialog box contain similar options. Use the VI Properties dialog box to set the style of controls and indicators in the current VI. Use the Front Panel page of the Options dialog box to set the style of controls and indicators in all new VIs.

Complete the following steps to change the style of the control or indicator LabVIEW creates when you right-click a terminal and select Create»Control or Create»Indicator from the shortcut menu in the current VI.

  1. Select File»VI Properties to display the VI Properties dialog box.
  2. Select Editor Options from the Category pull-down menu.
  3. Select a style from the Control Style for Create Control/Indicator options.
  4. Click the OK button.
Note  When you right-click a subVI terminal and select Create»Control or Create»Indicator from the shortcut menu, LabVIEW creates a control or indicator in the style of the subVI front panel object that corresponds to the subVI terminal. When you right-click a control or indicator and select Create»Control or Create»Indicator from the shortcut menu, LabVIEW creates a control or indicator in the style of the control or indicator you right-clicked.
