Configuring String Controls and Indicators

To configure a string constant, control, or indicator, right-click the object and select from the following shortcut menu items:

Normal Display

Backslash ('\') Codes Display

Password Display

Hex Display

Limit to Single Line

Update Value while Typing

Enable Wrapping

You also can right-click a string control or indicator and select Visible Items»Display Style from the shortcut menu to display a glyph on the object that indicates the display type.
