Validating Express VIs

You can run validation tests to detect common problems in Express VIs that you create. Complete the following steps to run validation tests on an Express VI.

  1. Select Tools»Advanced»Create or Edit Express VI to display the Create or Edit Express VI dialog box.
  2. Select the Express VI you want to validate from the Express VIs list.
  3. Click the Validate button to display the Validation dialog box. The Validation dialog box displays a list of test results for the Express VI. A green checkmark indicates the Express VI passed the corresponding test. A red X indicates the Express VI failed the test.
  4. Click a specific test name to view a description of the test.
  5. Click the OK button to return to the Create or Edit Express VI dialog box.
