Making a Control Visible on All Pages of a Tab Control

You can add a control to multiple pages of a tab control that will be visible on all the pages of the tab control.

Complete the following steps to add a control to multiple pages of a tab control.

  1. Create a multi-paged tab control in the front panel window.
  2. Create a control in an empty space on the front panel.
  3. Select the control you just created. Using the arrow keys, move the control onto the tab control.
Note  The control will display a drop shadow while the VI is in edit mode, indicating that it is not actually contained by the tab control. The drop shadow will not be displayed when the VI is running.
