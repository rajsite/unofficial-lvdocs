Creating New Documents in an ActiveX Container

Complete the following steps to create a new ActiveX document in an ActiveX container.

  1. Add an ActiveX container to the front panel window.

     Add  Find
  2. Right-click the ActiveX container and select Insert ActiveX Object from the shortcut menu to display the Select ActiveX Object dialog box.
  3. Select Create Document from the pull-down menu in the Select ActiveX Object dialog box.
  4. Select the document type from the list of those registered with the computer and click the OK button. The ActiveX document appears in the ActiveX container.
