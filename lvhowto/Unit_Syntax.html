Unit Syntax

Use the following syntax when you associate units with a numeric control or indicator:

  • Any unit can have any SI prefix. The prefix applies only to the unit directly after it.
  • Use a space to indicate multiplication. LabVIEW interprets multiple spaces as a single space.
  • Use a forward slash (/) to indicate division. If you use spaces around the forward slash, LabVIEW ignores the spaces. The forward slash applies only to the unit directly after it.
  • Use a carat (^) to indicate exponentiation. Exponents can be positive or negative integers, except zero (0). The carat applies only to the unit directly before it.

The following table displays examples of unit strings with correct syntax and the mathematical equivalent.

Unit String Mathematical Equivalent
ft lb ft * lb
m/s m / s (meters divided by seconds)
m /s/s m / (s * s)
s^2 s^2
kg m/s kg * m * s^–1
