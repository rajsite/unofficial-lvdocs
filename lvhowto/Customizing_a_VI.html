Customizing VIs

LabVIEW provides several options for customizing VIs. Select File»VI Properties or complete the following steps to access the options for customizing VIs.

  1. Right-click the icon in the upper-right corner of the front panel or block diagram.
  2. Select VI Properties from the shortcut menu.

If you want to change subVI settings during a specific call to that subVI, use the SubVI Node Setup dialog box.

You also can set VI properties programmatically.
