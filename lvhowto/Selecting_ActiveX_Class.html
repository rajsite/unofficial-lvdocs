Selecting an ActiveX Class

Complete the following steps to select an ActiveX server class.

  1. Add the Automation Refnum control to the front panel window.

     Add  Find
  2. Right-click the Automation Refnum control and select Select ActiveX Class»Browse from the shortcut menu.
  3. Select an ActiveX server class from the Select Object From Type Library dialog box.

After you select an ActiveX server class, you can invoke a method on that object or set a property for that object.
