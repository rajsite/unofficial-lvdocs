Converting Data to the Dynamic Data Type

Complete the following steps to convert numeric, Boolean, waveform, and array data to dynamic data for use with Express VIs.

  1. Add the Convert to Dynamic Data Express VI to the block diagram.

     Add  Find
  2. In the configuration dialog box that appears, specify the kind of data you want to convert to dynamic data and click the OK button.
  3. Wire the data you want to convert to the input of the Convert to Dynamic Data Express VI.
  4. Wire the output of the Convert to Dynamic Data Express VI to another Express VI, VI, or function that accepts dynamic data.
Note  Some VIs, such as VIs converted from Express VIs, and functions, such as the Merge Signals and Split Signals functions, accept dynamic data as an input.
