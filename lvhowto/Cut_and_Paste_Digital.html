Cutting, Copying, and Pasting Digital Data

Complete the following steps to cut digital data.

  1. Select the row or column in the digital data control or digital waveform control you want to cut.
  2. Right-click the row or column and select Data Operations»Cut Data from the shortcut menu.

Complete the following steps to copy digital data.

  1. Select the area in the digital data control or digital waveform control you want to copy.
  2. Right-click the area and select Data Operations»Copy Data from the shortcut menu.

Complete the following steps to paste digital data.

  1. Select an area in the digital data control or digital waveform control that consists of the same number of rows or columns you copied.
  2. Right-click the area and select Data Operations»Paste Data from the shortcut menu.

You can cut only whole rows or columns of data. You cannot create a new row or column with the digital data you cut. You must paste digital data into an area that is the same dimension as the area you cut or copied from. For example, if you copy four bits of data from one row, you must select four existing bits of data to paste over in the same row or in a different row. If you copy four bits of data from an area of two rows by two columns, you must paste the data into an area of two rows by two columns.
