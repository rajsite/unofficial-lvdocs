Adding or Removing an Instance in a Polymorphic VI

Complete the following steps to add an instance to a polymorphic VI you created.

Note  Use caution when editing polymorphic VIs that ship with LabVIEW.
  1. Right-click a polymorphic VI on the block diagram and select Open Polymorphic VI from the shortcut menu to display the Polymorphic VI window. You also can select File»Open and open a polymorphic VI to display the window.
  2. Click the Add button to display a file dialog box.
  3. Select a VI to include as an instance in the polymorphic VI and click the OK button. The VI you add must have the same connector pane pattern as all other instances of the polymorphic VI.
  4. (Optional) To change the order of the instances, select an instance in the list and click the up arrow or down arrow button. You also can select an instance in the list and drag it to a new position in the list.
  5. Select File»Save to save the VI.
  6. Select File»Close to close the Polymorphic VI window.

Complete the following steps to remove an instance from a polymorphic VI you created.

  1. Right-click a polymorphic VI on the block diagram and select Open Polymorphic VI from the shortcut menu to display the Polymorphic VI window. You also can select File»Open and open a polymorphic VI to display the window.
  2. Select the instance you want to remove from the Instance VIs list.
  3. Click the Remove button.
  4. Select File»Save to save the VI.
  5. Select File»Close to close the Polymorphic VI window.
