Deleting Attributes from Variant Data

Complete the following steps to delete an attribute(s) from variant data.

  1. Add the Delete Variant Attribute function to the block diagram.

     Add  Find
  2. Wire variant data with attributes to the variant input of the Delete Variant Attribute function.
  3. To delete all the attributes, leave the name (all attributes) input of the Delete Variant Attribute function unwired.

    To delete a specific attribute, enter the name of the attribute in the name (all attributes) input.
  4. Run the VI.
