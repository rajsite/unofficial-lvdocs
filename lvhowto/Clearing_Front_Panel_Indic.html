Clearing Front Panel Indicators

Complete the following steps to clear all front panel indicators automatically each time you run a top-level VI or call a subVI.

  1. Select File»VI Properties to display the VI Properties dialog box.
  2. Select Execution from the Category pull-down menu at the top of the VI Properties dialog box.
  3. Place a checkmark in the Clear indicators when called checkbox.
  4. Click the OK button.
