Configuring Your LabVIEW User Name

Complete the following steps to set or change your LabVIEW user name.

  1. Select Tools»User Name to display the User Login dialog box.
  2. By default, the operating system user name is selected. To change your LabVIEW user name, select a name from the pull-down menu. You also can select other from the pull-down menu and type a new user name in the Username text box.
  3. Click the OK button.
