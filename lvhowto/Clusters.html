Clusters

Use clusters to group data elements of a mixed type that logically belong together from other controls and indicators. An example of a cluster is the LabVIEW error cluster, which combines a Boolean value, a numeric value, and a string. A cluster is similar to a record or a struct in text-based programming languages.
