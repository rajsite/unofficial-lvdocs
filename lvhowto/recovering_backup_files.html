Recovering Backup Files Manually

The Automatic Recovery Error dialog box appears when LabVIEW encounters an error and cannot recover automatically saved files.

Complete the following steps to attempt to recover the backup files manually.

  1. Copy the backup files to a safe location on your computer.
  2. Click the Delete Backup Files button to remove the backups from the LabVIEW default data directory and continue launching LabVIEW.
  3. In LabVIEW, open the files you copied in step 1 by selecting File»Open and navigating to the copied files.
