Accessing Recently Opened Files

Complete the following steps to find recently opened files.

  1. Select File»Recent Files and select a filename (.vi, .ctl, .vit, .ctt) from the submenu that appears. Select File»Recent Projects to find recently opened project files (.lvproj).
  2. Click the filename to open it.

The files are listed in chronological order, with the most recently opened file listed first. If the lists are empty, the Recent Files and Recent Projects menu items are dimmed.

When two or more files have the same filename and reside in different directories, the Recent Files or Recent Projects list displays the full path.
