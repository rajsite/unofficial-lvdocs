Editing Existing Property and Invoke Nodes

You can use the Select Property or Select Method dialog box to edit an existing Property Node or Invoke Node.

Complete the following steps to edit a Property Node or Invoke Node.

  1. Right-click a property on a Property Node or a method on an Invoke Node and select Select Property»Browse or Select Method»Browse from the shortcut menu. The Select Property or Select Method dialog box appears.
  2. From the Properties or Methods and Dottable Properties list of the dialog box, select a property or method to replace the one currently shown in the Property Node or Invoke Node. The property or method you select appears in the Selected property or Selected method box.
  3. Click the OK button to update the node on the block diagram.
