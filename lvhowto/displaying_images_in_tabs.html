Displaying Images in Tab Controls

Complete the following steps to configure a tab control to display an image.

  1. Import an image to LabVIEW by selecting Edit»Import Picture to Clipboard.
  2. Set the tab layout to display an image by right-clicking the tab and selecting Advanced»Tab Layout»Image Only.
    Note  You can set the tab layout to display an image and text by selecting Advanced»Tab Layout»Image – Text.
  3. Import the image from the clipboard to the tab by right-clicking the tab and selecting Advanced»Tab Layout»Import Image from Clipboard.

You can remove an image from a tab by right-clicking the tab and selecting Advanced»Tab Layout»Remove Image from Tab.
