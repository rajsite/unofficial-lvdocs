Deleting Wires

Complete the following steps to delete a portion or all of a wire.

  1. Use the Positioning tool to single-click the wire to select one segment of the wire, double-click to select a branch, or triple-click to select the entire wire.
  2. Press the <Delete> key or the <Backspace> key.

You also can right-click a wire branch and select Delete Wire Branch from the shortcut menu to delete the wire branch.

Note  You can change the path of an existing wire without first deleting the wire. Create the new wire path between the same terminals, and LabVIEW deletes the previous wire path.
