Finding Property Nodes Linked to an Object or Terminal

Complete the following steps to find Property Nodes linked to an object or terminal.

  1. Right-click the control, indicator, or terminal.
  2. Select Find»Property Nodes from the shortcut menu. This appears only if you have multiple Property Nodes.
  3. Select the Property Node from the Search Results window and click the Go To button. LabVIEW highlights the Property Node.
