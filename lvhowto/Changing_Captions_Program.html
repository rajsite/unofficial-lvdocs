Changing Captions Programmatically

Complete the following steps to change a caption programmatically.

  1. Add a numeric control to the front panel window.

     Add  Find
  2. Right-click the control and select Visible Items»Caption from the shortcut menu.
  3. On the block diagram, right-click the control, select Create»Property Node»Caption from the shortcut menu, and select an option from the list that appears.

You also can use Property Nodes to show, hide, and change the caption programmatically.
