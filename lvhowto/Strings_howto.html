Strings

Strings group sequences of ASCII characters. Strings provide a platform-independent format for information and data. LabVIEW stores a string in a special format in which the first four bytes of the array of characters form a 32-bit signed integer that stores how many characters appear in the string. Thus, a string with n characters requires n + 4 bytes to store in memory.
