Distributing Objects

Use the Distribute Objects pull-down menu ( ) or select Edit»Distribute Items to space objects evenly. The distribution option you select remains in effect until you change it. A dark border around the icon on the palette indicates the current distribution option.

Complete the following steps to distribute objects.

  1. Select the objects you want to distribute.
  2. Select the Distribute Objects pull-down menu, located to the right of the Align Objects pull-down menu on the toolbar in the front panel and block diagram windows.
  3. Select from the following options:
    • Top Edges—Distributes the selected objects so that the top borders are spaced equally apart.
    • Vertical Centers—Distributes the selected objects so that the vertical centers are spaced equally apart.
    • Bottom Edges—Distributes the selected objects so that the bottom edges are spaced equally apart.
    • Vertical Gap—Distributes the selected objects so that the vertical space between the objects is equally sized.
    • Vertical Compress—Compresses the selected objects so that there is no space between the top and bottom borders of the objects.
    • Left Edges—Distributes the selected objects so that the left edges are spaced equally apart.
    • Horizontal Centers—Distributes the selected objects so that the horizontal centers are spaced equally apart.
    • Right Edges—Distributes the objects so that the right edges are spaced equally apart.
    • Horizontal Gap—Distributes the selected objects so that the horizontal space between the objects is equally sized.
    • Horizontal Compress—Compresses the selected objects so that there is no space between the left and right borders of the objects.

To repeat a distribution operation on another group of objects, press the <Ctrl-D> keys. (macOS) Press the <Command-D> keys. (Linux) Press the <Alt-D> keys.

You also can select Edit»Distribute Items to repeat a distribution operation on another group of objects.
