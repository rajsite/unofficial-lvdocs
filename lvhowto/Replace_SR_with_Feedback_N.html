Replacing Shift Registers with a Feedback Node

Complete the following steps to replace shift registers with a Feedback Node.

  1. Right-click the shift register.
  2. Select Replace with Feedback Node from the shortcut menu.

Replacing a Feedback Node with Shift Registers

Complete the following steps to replace a Feedback Node with shift registers.

  1. Right-click the Feedback Node.
  2. Select Replace with Shift Register from the shortcut menu.
