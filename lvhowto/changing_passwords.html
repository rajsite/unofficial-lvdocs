Changing Passwords (Windows)

Complete the following steps to change the password for the current (logged in) user.

  1. Select Tools»Security»Change Password to display the NI Security Change Password dialog box.
  2. In the Old password text box, enter the password for the current (logged in) user.
  3. In the New password text box, enter the new password for the user.
  4. In the Confirm password text box, re-enter the new password for the user.
  5. Click the OK button to change password.
