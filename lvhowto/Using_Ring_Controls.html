Using Ring Controls

Use a ring control to select an item from a list without having to know what value that item represents. LabVIEW passes the value associated with the selected item to the block diagram. Select a case from a Case structure (conditional code) that carries out the selected item.

Complete the following steps to create a ring control.

  1. Add a ring control to the front panel.

  2. Add text or picture items to the ring control.
  3. (Optional) Remove items from the ring control.
