Adding Cases between Case Structure Subdiagrams

Complete the following steps to add a case between existing case subdiagrams in a Case structure:

  1. Add a new case at the end of the order.
  2. With the new case visible, right-click the structure border and select Rearrange Cases from the shortcut menu.
  3. Select the position for the new case. The subsequent subdiagrams shift down in the order.
  4. Click the OK button to save the changes.
Tip  You also can press the <Shift-Enter> keys in the selector label to add a case to a Case structure.

You also can shift existing cases down in the order. Refer to the Case Structure - Selector Data Types VI in the labview\examples\Structures directory for an example of using Case structures.
