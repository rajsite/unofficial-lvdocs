Setting the Alignment Grid Size for the Current VI

Use the VI Properties dialog box to set the size of the alignment grid for the current VI. Use the grid options on the Front Panel page and the Block Diagram page of the Options dialog box to set the size of the grid for all new VIs.

Complete the following steps to set the size of the grid for the current VI.

  1. Select File»VI Properties to display the VI Properties dialog box.
  2. Select Editor Options from the Category pull-down menu.
  3. Enter the size to use for the front panel and block diagram grid in pixels in the Front panel and Block diagram fields, respectively.
