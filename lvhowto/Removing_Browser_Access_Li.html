Removing Browser Access List Entries

Complete the following steps to remove a browser address from the Browser Access List.

  1. Select Tools»Options to display the Options dialog box.
  2. Select Web Server from the Category list.
  3. In the Browser Access List in the Browser Access section, select the address you want to remove.
  4. Click the Remove button.
  5. Click the OK button to close the Options dialog box and save the changes.
