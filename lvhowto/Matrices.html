Matrices

Use matrices to group rows or columns of real or complex scalar data for some math operations, such as linear algebra operations. A real matrix contains double-precision elements, and a complex matrix contains complex elements with double-precision components. A matrix can have only two dimensions. You cannot create an array of matrices. Use the Bundle function to combine two or more matrices and create a cluster. Matrices have the same restrictions as arrays.
