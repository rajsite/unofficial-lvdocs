Removing Machine Access List Entries

Complete the following steps to remove a browser address from the Machine access list.

  1. Select Tools»Options to display the Options dialog box.
  2. Select VI Server from the Category list.
  3. In the Machine access section, select the address in the Machine access list you want to remove.
  4. Click the Remove button.
  5. Click the OK button to close the Options dialog box and save the changes.
