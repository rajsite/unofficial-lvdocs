Adding Header Text to Listboxes, Tables, and Tree Controls

Complete the following steps to add header text to a listbox, table, or tree control.

  1. Right-click the control and select Visible Items»Row Headers and Visible Items»Column Header, or Visible Items»Column Headers for controls with multiple columns, from the shortcut menu to display the row and column headers.
Note  Single-column listboxes do not have row headers.
  1. Use the Labeling tool to click the header in which you want to add text.
  2. Enter the text you want.
Note  You cannot enter text in the top left header cell.
