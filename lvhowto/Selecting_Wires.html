Selecting Wires

Use the Positioning tool to single-click the wire to select one segment of the wire, double-click to select a branch, or triple-click to select the entire wire.

To add a wire to a group of selected wires, press the <Shift> key while you select the new wire.

Use the Positioning tool to drag a selection rectangle around the parts of the wire you want to select.

After you select the wire, you can delete it or move it.
