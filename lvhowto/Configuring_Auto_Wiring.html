Configuring Automatic Wiring

Complete the following steps to enable or configure automatic wiring.

  1. Select Tools»Options to display the Options dialog box.
  2. Select Block Diagram from the Category list.
  3. Place a checkmark in the Enable auto wiring checkbox to enable automatic wiring.
  4. Edit the Minimum distance and Maximum distance to set the minimum distance in pixels objects must have between them to wire automatically and the maximum distance in pixels objects can be separated and still wire automatically.
