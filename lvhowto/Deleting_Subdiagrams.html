Deleting Subdiagrams

Complete the following steps to delete the visible subdiagram.

  1. Move to the subdiagram you want to delete.
  2. Right-click the structure border and select one of the following options:
    • Delete This Case
    • Delete This Frame
    • Delete This Subdiagram
    • Delete This Event Case

If only one subdiagram exists, you cannot delete the visible subdiagram.
