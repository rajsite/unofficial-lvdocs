Displaying and Hiding the Overflow Status LED for Fixed-Point Numbers

After you configure a fixed-point control, constant, or indicator to include an overflow status, you can display and hide an overflow status LED on the control, constant, or indicator. This LED indicates whether the fixed-point number is the result of an operation that overflowed. Complete the following steps to display or hide an overflow status LED.

  1. Configure the fixed-point control, constant or indicator to include an overflow status.
  2. Right-click the fixed-point control or indicator from the front panel, or right-click the fixed-point constant from the block diagram.
  3. Select Visible Items»Overflow Status LED from the shortcut menu.
Note  If you do not configure the control, constant, or indicator to include an overflow status, the Overflow Status LED shortcut menu item does not appear.
