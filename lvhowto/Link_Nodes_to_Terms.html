Linking Property Nodes and Invoke Nodes to Terminals

If you copy or clone a Property Node or Invoke Node, LabVIEW can disconnect the node from its front panel object.

Complete the following steps to link a Property Node or Invoke Node to a front panel object.

  1. Right-click the node.
  2. Select Link to and the name of an object from the shortcut menu. LabVIEW replaces the node with an implicitly linked node.
  3. Right-click the node again and select Disconnect From Control from the shortcut menu to remove the implicitly linked node.

You also can use the Link to shortcut menu to associate an implicitly linked node with a different front panel object. The Link to shortcut menu includes an option for each front panel object in the VI.
