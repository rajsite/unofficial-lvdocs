Merging Signals

Use the Merge Signals function to combine the data from two or more different acquired or simulated signals into one wire. You can wire the combined signals to a graph with multiple plots or save them to a file.

Complete the following steps to merge signals before you wire the signals to the input of another VI, function, graph, or chart.

  1. Add the Merge Signals function to the block diagram.

     Add  Find
  2. Resize the Merge Signals function so it includes enough inputs for the signals you want to wire.
  3. Wire each signal to an input signal input.
  4. Wire the combined signal output to the signals input of another VI, function, graph, or chart.
Note  You can split any combined signals into the individual signals.
