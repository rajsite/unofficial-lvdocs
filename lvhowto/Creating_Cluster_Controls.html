Creating Cluster Controls and Indicators

Create a cluster front panel control or indicator by combining a cluster shell with a valid data object or element, such as Boolean, string, graph, array, numeric, and so on.

Complete the following steps to create a cluster control or indicator.

  1. Add a cluster shell to the front panel window.

     Add  Find
  2. If the Controls palette is not already visible, right-click the element display or the front panel window to display the Controls palette.
  3. Drag a control or indicator into the cluster shell. For example, drag a numeric control into the cluster shell.

You also can copy or drag an existing cluster constant from the block diagram window to the front panel window to create a cluster control.
