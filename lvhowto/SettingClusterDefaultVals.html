Setting Cluster Default Values

Complete the following steps to set the default value of the entire cluster to the current value of each individual element.

  1. In the front panel window, right-click the cluster shell border.
  2. Select Data Operations»Make Current Value Default from the shortcut menu.

Complete the following steps to reset all elements to their individual default configurations

  1. In the front panel window, right-click the cluster shell border.
  2. Select Data Operations»Reinitialize to Default Value from the shortcut menu.

Complete the following steps to change the default value of a single element in a cluster.

  1. In the front panel window, select the element whose default value you want to set.
  2. Enter the new default value for the element.
  3. Right-click the element and select Data Operations»Make Current Value Default from the shortcut menu.
