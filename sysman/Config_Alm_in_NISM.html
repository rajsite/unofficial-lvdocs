Configuring Alarms for Shared Variables (DSC Module)

Complete the following steps to configure alarms for shared variables.

  1. In the system tree pane, locate the alarm you want to configure and expand this alarm. The System Manager displays the attributes of the alarm.
  2. Right-click the attribute you want to configure and select Probe from the shortcut menu. A Probe view of the attribute appears.
  3. Specify a new value of the attribute in the New Value text box or option buttons to configure this attribute.
    Note  You cannot configure the alarm statuses, such as Ack, Description, and Set.
  4. Click the Set button to set the new value of the attribute.
Note  You also can configure alarms by using the Auto View.
