Observing Trend View of Variables

Use the Variable Probe view in the System Manager to monitor the value of a variable over time using the trend view graph.

Complete the following steps to monitor the value of a variable over time using the trend view graph.

  1. Right-click a variable in the system tree.
  2. Select Probe from the shortcut menu to open a new Variable Probe view for the selected variable.
  3. Place a checkmark in the Show Trend checkbox to display the trend view graph.
