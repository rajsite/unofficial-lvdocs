Help Menu

The Help menu contains items to explain and define System Manager features and provide System Manager documentation.

  • System Manager Overview—Displays the NI Distributed System Manager Help. Use this help file for information about creating and managing shared or network variables and other system tree objects.
    Note  Refer to the LabVIEW Help, NI Measurement Studio Help, or LabWindows/CVI Help for more information about using variables created with the System Manager in your applications.
  • About System Manager—Displays general information about the current installation, such as the version number.
