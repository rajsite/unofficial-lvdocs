Registering Remote Computers

You need to register remote computers before setting up networked applications. Refer to the LabVIEW Help for more information about setting up networked applications. Complete the following steps to register a computer.

  1. In the NI Distributed System Manager, select Actions»Register Computer to display the Register Computer dialog box.
  2. In the Computer Name text box, specify the name of a computer.
  3. Click the OK button to register this computer.

After you register a remote computer, the System Manager displays this registered computer under the Network Items folder in the system tree pane.

Note  You can right-click a remote computer under the Network Items folder and select Unregister Computer to unregister a remote computer.
