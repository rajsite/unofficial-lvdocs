Creating Processes

You can create processes on a local or LabVIEW Real-Time target.

Complete the following steps to create a process with the System Manager.

  1. Right-click a system in the system tree.
  2. Select Add Process from the shortcut menu to display the Add Process dialog box.
  3. Configure the process in the Add Process dialog box. You can use this dialog box to configure the process name and whether the process will run after you create it.

After you create a process, you can add shared or network variables or add I/O servers to that process.
