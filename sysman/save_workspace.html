Saving and Restoring a System Manager Workspace

When you customize the System Manager workspace, you can save the customized workspace and restore it in a later System Manager session.

Complete the following steps to save and restore a customized System Manager workspace.

  1. Customize the System Manager workspace.
  2. Select File»Save Workspace to save your workspace as a .smw file.
  3. To restore a workspace in a later session, select File»Open Workspace and navigate to the .smw file you want to restore.
